Pepe Coin Sentiment & Price Analysis
This project explores how public sentiment on Twitter correlates with the price of Pepe Coin over the past month.

Key Insight
We found a negative correlation between public sentiment and price. Interestingly, bearish sentiment often preceded price spikes, suggesting a potential contrarian edge in meme coin psychology.
This reflects typical retail-driven panic/FOMO cycles — and hints at opportunities for sentiment-aware trading strategies.

Methodology
Scraped 200+ tweets using snscrape (or used preloaded data)

Performed sentiment analysis with TextBlob

Collected price data from CoinGecko

Merged and visualized data to reveal correlations

Files
pepe_price.csv: Historical price data

pepe_tweets.csv: Tweet content with timestamps

pepe_sentiment_insights.csv: Combined sentiment and price data with analysis columns

Why This Matters
This analysis suggests that social sentiment may act as a leading indicator in meme coin markets. By quantifying hype and fear, traders might anticipate market behavior in ways traditional tools can't.
